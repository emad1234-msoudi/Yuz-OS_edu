🐆 Yuz-OS Edu

سبک، سریع و باهوش؛ مثل یوز

یک پروژه آموزشی متن‌باز برای مدارس، دانش‌آموزان و معلمان که با هدف ایجاد یک محیط آموزشی یکپارچه، کم‌هزینه و قابل اجرا روی رایانه‌های قدیمی توسعه داده شده است.

---

📖 درباره پروژه

Yuz-OS Edu یک سیستم‌عامل جدید از صفر نیست؛ بلکه یک پروژه آموزشی مبتنی بر Debian 13 است که تلاش می‌کند با استفاده از نرم‌افزارهای آزاد و متن‌باز، یک محیط آموزشی آماده و یکپارچه را در اختیار مدارس قرار دهد.

ایده اصلی پروژه از یک مسئله واقعی شکل گرفت:

بسیاری از مدارس به دلیل هزینه بالای تجهیز آزمایشگاه‌های رایانه، محدودیت بودجه و قدیمی بودن سخت‌افزارها، امکان استفاده گسترده از ابزارهای آموزشی دیجیتال را ندارند.

Yuz-OS Edu تلاش می‌کند این مشکل را با استفاده مجدد از رایانه‌های موجود و فراهم کردن یک محیط آموزشی آماده، ساده‌تر کند.

---

🎯 اهداف پروژه

- فراهم کردن محیط آموزشی یکپارچه برای مدارس
- افزایش دسترسی به ابزارهای آموزشی دیجیتال
- کاهش هزینه‌های تجهیز و نگهداری
- استفاده مجدد از رایانه‌های قدیمی
- کاهش وابستگی به اینترنت
- ساده‌سازی نصب و راه‌اندازی نرم‌افزارهای آموزشی
- ایجاد بستری مناسب برای یادگیری علوم، برنامه‌نویسی و فناوری
