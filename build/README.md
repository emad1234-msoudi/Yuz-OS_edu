🟦 Yuz-OS

An Educational & Optimized Linux Operating System for Students

Yuz-OS is a lightweight, stable, and educational Linux distribution based on Debian 13 (Trixie), designed specifically for students, schools, and learning environments.

The main goal of Yuz-OS is to provide a ready-to-use system that runs smoothly even on low-resource hardware, while offering essential tools for programming, science, and general education.


---

🎯 Project Goals

Provide a stable and reliable educational operating system

Ensure smooth performance on low-end systems (4 GB RAM, dual-core CPU)

Offer a clean desktop with no unnecessary or distracting software

Use official Debian technologies and standards

Be easy to install, easy to maintain, and easy to explain (important for academic evaluation)



---

⚙️ Why Debian?

Yuz-OS is built on Debian, one of the most respected Linux distributions in the world:

🛡️ Proven stability and security

🌍 Widely used in education and research

📦 Huge and well-maintained software repositories

🔧 Excellent base for long-term projects

🧠 Perfect for teaching real Linux concepts, not shortcuts


This makes Debian an ideal foundation for an educational operating system.


---

🚀 Performance & Optimization

Yuz-OS is carefully optimized for weak and old systems:

✅ ZRAM enabled by default to reduce RAM pressure

✅ Fast boot on both UEFI and Legacy BIOS

✅ Clean startup (no unnecessary background services)

✅ Optimized package selection


> Yuz-OS has been tested successfully on systems with 4 GB RAM, 2 CPU cores, and 35 GB disk space.
